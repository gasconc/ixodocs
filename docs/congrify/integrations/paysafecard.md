## [Paysafecard](https://docs.congrify.com/#/Integrations/paysafecard?id=paysafecard)
## [About PaySafecard](https://docs.congrify.com/#/Integrations/paysafecard?id=about-paysafecard)
Paysafecard () is a prepaid online payment solution that allows users to make secure transactions without the need for a bank account or credit card. It works through vouchers with a unique PIN, which can be purchased at retail locations and used for online payments across gaming, entertainment, and e-commerce platforms. Paysafecard is widely accepted and ensures privacy and security by eliminating the need to share financial details.
## [How to connect with Paysafecard](https://docs.congrify.com/#/Integrations/paysafecard?id=how-to-connect-with-paysafecard)
To connect with Paysafecard you would need to obtain their API credentials. If you need guidance, please contact their support. 
## [Required access credentials](https://docs.congrify.com/#/Integrations/paysafecard?id=required-access-credentials)
  1. Account Name The name of Paysafecard's Account
  2. API Key The API Key of Paysafecard's merchant account
  3. Merchant ID The Merchant ID of Paysafecard's merchant account

## [Importing Historical Data](https://docs.congrify.com/#/Integrations/paysafecard?id=importing-historical-data)
During the onboarding process with Congrify, you can import historical data from PaySafecard. Please reach out to Congrify's team to confirm which data can be historically downloaded.
