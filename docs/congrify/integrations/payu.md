## [Payu](https://docs.congrify.com/#/Integrations/payu?id=payu)
## [About Payu](https://docs.congrify.com/#/Integrations/payu?id=about-payu)
Payu () is a global fintech company providing online payment solutions for businesses and consumers. It offers a wide range of payment methods, including credit/debit cards, bank transfers, and local payment options, catering to emerging markets. PayU enables seamless and secure transactions with fraud prevention tools and easy integration for e-commerce platforms, making it a key player in digital payments worldwide.
## [How to connect with PayU](https://docs.congrify.com/#/Integrations/payu?id=how-to-connect-with-payu)
To connect with Payu you would need to obtain their SFTP credentials. If you do not have these already then a Username and Private key need to be requested from PayUto be able to retrieve your reports from their SFTP server for each Merchant Account.
## [Required access credentials](https://docs.congrify.com/#/Integrations/payu?id=required-access-credentials)
  1. Account Name Name of the PayU Account 
  2. Username The username used for connection to the PayU SFTP server
  3. Private Key The contents of the private key used for authentication when connecting to the PayU SFTP server
  4. Directory List The list of paths on the PayU SFTP where reports can be found and downloaded

## [Importing Historical Data](https://docs.congrify.com/#/Integrations/payu?id=importing-historical-data)
During the onboarding process with Congrify, you can import historical data from Payu. Please reach out to Congrify's team to confirm which data can be historically downloaded.
## [Available Reports](https://docs.congrify.com/#/Integrations/payu?id=available-reports)
The following reports are available for the PaySafecard integration and will provide the necessary data for a successful integration with Congrify. Please ensure that you select these reports as a minimum when configuring your PaySafecard connection:
  * Dispute Report
  * Payout Report
  * Settlement Report
  * Transaction Report
