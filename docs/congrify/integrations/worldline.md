## [Worldline](https://docs.congrify.com/#/Integrations/worldline?id=worldline)
## [About Worldline](https://docs.congrify.com/#/Integrations/worldline?id=about-worldline)
Worldline () is a global leader in payment services, headquartered in France, with operations spanning Europe, Asia, and the Americas. Serving over **1 million merchants** worldwide, Worldline provides secure and innovative payment processing solutions for businesses of all sizes. It offers integrated solutions for in-store, online, and mobile transactions, along with advanced fraud prevention, reporting tools, and dedicated customer support. With a strong focus on security, scalability, and digital transformation, Worldline helps businesses optimize their payment systems and deliver seamless transaction experiences.
## [How to connect with Wordline](https://docs.congrify.com/#/Integrations/worldline?id=how-to-connect-with-wordline)
To connect with Worldline you need to have a SFTP (Secure File transfer protocol) setup active with Worldline. If you do not have these already then a Username and Password combination must be requested from Worldline to be able to retrieve your reports from their SFTP server for each Merchant Account.
## [Required access credentials](https://docs.congrify.com/#/Integrations/worldline?id=required-access-credentials)
  1. Account Name The name of the Worldline Account
  2. Username The username used for connection to the Worldline SFTP server. 
  3. Password The password used for authentication when connecting to the Worldline SFTP server.

## [Importing Historical Data](https://docs.congrify.com/#/Integrations/worldline?id=importing-historical-data)
During the onboarding process with Congrify, you can import historical data from Wordlines. Please reach out to Congrify's team to confirm which data can be historically downloaded, based on the SFTP data retention configuration.
## [Available Reports](https://docs.congrify.com/#/Integrations/worldline?id=available-reports)
The following reports are available for the Worldline integration and will provide the necessary data for a successful integration with Congrify. Please ensure that you select these reports as a minimum when configuring your Wordline connection:
  * Transaction report
  * Financial report
