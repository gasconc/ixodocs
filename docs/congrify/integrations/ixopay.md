## [ixopay](https://docs.congrify.com/#/Integrations/ixopay?id=ixopay)
## [About Ixopay](https://docs.congrify.com/#/Integrations/ixopay?id=about-ixopay)
IXOPAY () is a leading payment orchestration platform that streamlines global online payments for enterprise-level businesses. Designed for scalability and flexibility, IXOPAY enables merchants to connect to hundreds of payment service providers and methods through a single, unified API.
It offers intelligent transaction routing, secure tokenization, PCI-compliant data management, and advanced fraud-prevention tools — ensuring optimized approval rates, lower processing costs, and enhanced reliability. IXOPAY also provides automated reconciliation, comprehensive reporting, and support for subscription billing, making it an ideal solution for international e-commerce companies, marketplaces, and recurring-revenue businesses.
## [How to connect with Ixopay](https://docs.congrify.com/#/Integrations/ixopay?id=how-to-connect-with-ixopay)
To integrate IXOPAY with Congrify, you will need to provide the following credentials:
  * Merchant Account Name
  * API Key
  * API Key Password

For detailed guidance on how to obtain these credentials, please refer to IXOPAY’s official user manual: 
## [Find your API Key & API Key Password](https://docs.congrify.com/#/Integrations/ixopay?id=find-your-api-key-amp-api-key-password)
To create your API Key and API Key Password in IXOPAY, follow these steps:
  1. Log in to the IXOPAY portal and navigate to System Setup → API Keys. 

![Alt text](https://docs.congrify.com/Images/ixopay_tenant_api_keys.png)
  2. Click New API Key, then select BDS (BI Data Source) under the Services section.

![Alt text](https://docs.congrify.com/Images/ixopay_create_new_api_keys.png)
  3. After creating the key, your API Key and API Key Password will be displayed. This is the only time the password will be visible, so make sure to securely store it before closing the window.

![Alt text](https://docs.congrify.com/Images/ixopay_Api_key_successful.png)
  4. Enter your credentials in the Congrify → Connections → IXOPAY Connection section.

## [Importing Historical Data](https://docs.congrify.com/#/Integrations/ixopay?id=importing-historical-data)
During onboarding, you may choose to import historical transaction data from IXOPAY into Congrify.To initiate this process, please contact our support team or your designated account representative.
