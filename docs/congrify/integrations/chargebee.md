## [Chargebee](https://docs.congrify.com/#/Integrations/chargebee?id=chargebee)
## [About Chargebee](https://docs.congrify.com/#/Integrations/chargebee?id=about-chargebee)
Chargebee () is a subscription management and recurring billing platform that helps businesses automate invoicing, payments, and revenue operations. It supports various payment gateways, tax compliance, and analytics, making it a popular choice for SaaS, eCommerce, and other subscription-based businesses. Chargebee integrates with multiple financial and CRM tools, streamlining the entire subscription lifecycle.
## [How to connect with Chargebee](https://docs.congrify.com/#/Integrations/chargebee?id=how-to-connect-with-chargebee)
To connect with Chargebee please contact Congrify's support.
## [Required access credentials](https://docs.congrify.com/#/Integrations/chargebee?id=required-access-credentials)
  1. Name The name of Chargebee's Account
  2. API Key The API Key on the Chargebee platform to ingest data for
  3. Site The URL of your Chargebee account.

## [Importing Historical Data](https://docs.congrify.com/#/Integrations/chargebee?id=importing-historical-data)
During the onboarding process with Congrify, you can import historical data from Chargebee. Please reach out to Congrify's team to confirm which data can be historically downloaded.
## [Available Reports](https://docs.congrify.com/#/Integrations/chargebee?id=available-reports)
The following reports are available for the Chargebee integration and will provide the necessary data for a successful integration with Congrify. Please ensure that you select these reports as a minimum when configuring your Chargebee connection:
  * Credit Note report
  * Invoice Report
  * Order Report
  * Subscription Report
  * Transaction Report
