## [User and Notification management](https://docs.congrify.com/#/Fintech/UserManagementNotifications?id=user-and-notification-management)
  1. Navigate to the 'Users and Notifications' tab under your User profile as an Administrator 

![Alt text](https://docs.congrify.com/Images/UserMN_1.png)
  2. Add new users to your account by clicking on ‘+ Invite New Users’ button, and inserting their email address and assigning a role based access as 'Merchant User' (i.e. providing read and limited write access to the Congrify analytics platform) and/ or as an 'Administrator' (i.e. extended account, connection and credentials management). The new Users are notified over email and can then access their User profile on Congrify to further personalize their User profile with their name, profile photo, and employee designation.

![Alt text](https://docs.congrify.com/Images/UserMN_2.png)
![Alt text](https://docs.congrify.com/Images/UserMN_3.png)
  3. As a privileged Administrator in the Users tab, you can search and view the current details, access rights (including toggling temporary access as an 'active’ user), user status (including MFA onboarding) and activity details of your onboarded team, and further manage their role-based access rights through the User Management page.

![Alt text](https://docs.congrify.com/Images/UserMN_4.png)
  4. Under the Notifications tab, you can also individually control the notifications, alerts and updates pushed to each User in the Congrify app and external 3rd party (such as Slack) channels. 

![Alt text](https://docs.congrify.com/Images/UserMN_5.png)
  5. Additionally, you can view all notifications and alerts from the Congrify team of upcoming maintenance directly on the dashboard 

![Alt text](https://docs.congrify.com/Images/UserMN_6.png)
## [Role Types Summary](https://docs.congrify.com/#/Fintech/UserManagementNotifications?id=role-types-summary)  
| Role Type  | Role Permissions  |  
| --- | --- |  
| Owner  | Is the owner of the account. When registering an account with Congrify, the first user is automatically assigned as an Owner. The Owner can see all functionalities and settings in Congrify and is the only user that cannot be deleted. If you need to delete yourself as an owner, please contact our support team.  |  
| Admin  | Has all the accesses in the platform to access user ('Merchant Users') management including access rights, manage connection credentials, set merchant settings, etc.  |  
| Merchant User  | Is the user profile who has access to the reporting, visualization and analytical data features of the Congrify platform, with limited personal management of user profile, settings, etc.  |  
## [Account Settings](https://docs.congrify.com/#/Fintech/UserManagementNotifications?id=account-settings)
  1. Customize your preferences for timezone, currency, decimal precision, default common terminology and change comparison directly through your 'Account Settings' under your User tab.

![Alt text](https://docs.congrify.com/Images/UserMN_7.png)
  2. As part of the account settings changes, you can also set the default rate calculations for Fees basis ‘Sales' or ‘Net Sales’ denominations. However, you can also toggle the rate calculations directly on the 'Analytics’-> Fees page to re-compute and compare such rates in real-time. 

![Alt text](https://docs.congrify.com/Images/UserMN_8.png) ![Alt text](https://docs.congrify.com/Images/UserMN_9.png) ![Alt text](https://docs.congrify.com/Images/UserMN_10.png)
  3. You can also toggle visibility of changes in aggregated data on the dashboard and analytics pages. By default the changes are computed on the same time period directly preceding the selected time, however, as a User, you can select custom change time-periods directly on each dashboard or analysis page to compare your payments data aggregated over different selected time periods vs change period through the UI.
a. First, toggle change computation through the User tab → Account Settings, selecting ‘Show Changes’ under 'Change Comparison'.
![Alt text](https://docs.congrify.com/Images/UserMN_11.png)
b. On the dashboard, the percentage increase or decrease of the data point (i.e. 'change’) should now be visible. 
![Alt text](https://docs.congrify.com/Images/UserMN_12.png)
c. You can further configure the comparison periods for the 'change' computation by selecting the time window for the change period directly on the dashboard.
![Alt text](https://docs.congrify.com/Images/UserMN_13.png)
