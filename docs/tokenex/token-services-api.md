# Token Services API
This page will help you get started with the **Token Services** API.
The TokenEx **Token Services** API is used to tokenize and detokenize sensitive data such as Social Security numbers, as well as manage your token vault(s).
Heads-up!
**TokenEx API v1** has been deprecated and is now in break/fix mode. All new enhancements are being added to [API v2](https://documentation.ixopay.com/modules/api/tokenex/token-services).
