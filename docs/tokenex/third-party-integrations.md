# Third Party Integrations
In this section of the documentation, you will find information on how to configure the iFrame to interact with supported third-party providers. These integrations allow you to leverage external services—such as fraud detection, risk analysis, and 3-D Secure authentication—seamlessly during the tokenization process. You will learn about the specific prerequisites, configuration parameters, and implementation steps required to enable these features within your iFrame implementation.
