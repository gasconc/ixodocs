# Network Tokenization API
The Network Tokenizaton API enables TokenEx to act as an On-Behalf-Of Token Requestor (OBOTR), which enables merchants, acquirers, and payment service providers to integrate with global Token Service Providers (Visa VTS, Mastercard MDES, and American Express AETS) using a single, unified API.
