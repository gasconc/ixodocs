# Universal Token Payment Orchestration
Beta Product Offering - Use Universal Tokens with dynamic payment flows and smart transaction routing.
Beta Product Offering
Usage of Universal Tokens with Payment Orchestration is limited to participants in the Beta program. Please contact your Customer Success Manager for more information.
Build [payment orchestration](https://www.ixopay.com/en/payment-orchestration) into your payments strategy by utilizing [IXOPAY](https://www.ixopay.com)'s catalog of [170+ adapters](https://www.ixopay.com/en/features/adapter-world) - to access a wide variety of payment gateways and payment service providers (PSPs) through a single, robust REST API using JSON.
Connect directly to an adapter via a connector or use a meta-connector to create a [resilient, multi-PSP cascade ](https://www.ixopay.com/en/features/smart-transaction-routing) for payments to flow through that is customized with rules specific to your payments strategy.
The following pre-requisites are required to use Universal Tokens with Payment Orchestration
  * Active Universal Token subscription
    * Configured vault and API key with Payment Services All Access permission
  * At least one IXOPAY connector

To get started, follow the steps below
  1. Request Payment Services access via Support ticket or your Customer Success Manager.
  2. Support will create your IXOPAY portal account and provision access.
  3. Create your merchant connectors to adapters within the IXOPAY portal.
  4. If using a meta-connector, configure transaction processing rules.
  5. Retrieve connector-specific authentication parameters from the IXOPAY portal.
  6. Follow the [Universal Token Transaction API reference](https://documenter.getpostman.com/view/11766591/2sA3e5dnbm).

Please reach out to your Customer Success Manager or Support for assistance!
