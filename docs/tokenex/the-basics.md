# Overview of TokenEx API v1
This page will help you get started with **TokenEx API v1**..
The TokenEx Web API accepts JSON payloads. All TokenEx API requests must utilize TLS 1.2 (HTTPS).
**Host:** test-api.tokenex.com  
**Port:** 443  
**Content Type:** application/json
Heads-up!
**TokenEx API v1** has been deprecated and is now in break/fix mode. All new enhancements are being added to [API v2](https://documentation.ixopay.com/modules/docs/tokenex/the-basics-1).
