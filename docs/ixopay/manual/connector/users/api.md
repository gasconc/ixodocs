# API User
## Video: Set Up an API User[​](https://documentation.ixopay.com/manual/docs/connector/users/api#video-set-up-an-api-user "Direct link to Video: Set Up an API User")
![](https://documentation.ixopay.com/manual/assets/ideal-img/preview-vimeo-api-user.9898bcd.1300.png)
Learn how to set up a new API User.
In order to be able to send Transactions via the [Transaction API](https://documentation.ixopay.com/api/transaction/transaction-api), you have to set up an API User per Merchant Profile. To do so follow these steps:
  1. Create a new (Merchant) User as described in the [Web User](https://documentation.ixopay.com/manual/docs/connector/users/web) section
  2. Select the **User Type** - API

Depending on the selected User Type, different configuration options are available when editing the User:
![\(Merchant\) User Overview](https://documentation.ixopay.com/manual/assets/ideal-img/merchant-user-overview.429adc7.1280.png)(Merchant) User Overview
## Edit API Users[​](https://documentation.ixopay.com/manual/docs/connector/users/api#edit-api-users "Direct link to Edit API Users")
For API Users (see Edit API User):
  1. **Disabled** : Checkbox to disable the user. This action prevents immediate access to the platform for this user after saving.
  2. **Settings** : Enable Fee API, Enable Push API, Enable CDE Login, Enable Snapshot Retrieval
  3. Enable/Disable **Reset Dashboard**
  4. Enable/Disable **Archived** : Archives this user. You can undo the archiving in the Archived Users section (see Archived (Merchant) Users)
  5. Click **Save User** to save the changes
