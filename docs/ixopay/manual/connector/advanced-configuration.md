# Advanced Configuration
In this section of the manual, you will find detailed guides on customizing connector behavior beyond the standard setup.

GST Fee Set up surcharge and GST fee calculation by linking fee entities to connectors and enabling the surcharge connector setting.](https://documentation.ixopay.com/manual/docs/connector/advanced-configuration/surcharge)## [ Subbrand Selection Enable subbrand selection on connectors so customers can choose a specific card subbrand on the Hosted Payment Page during checkout.](https://documentation.ixopay.com/manual/docs/connector/advanced-configuration/subbrand-selection)## [ Field Operation Apply the Field Operation connector setting to truncate, pad, or transform transaction data fields before processing or storing them.](https://documentation.ixopay.com/manual/docs/connector/advanced-configuration/field-operation)## [ Field Expression Operation Use the Field Expression Language connector setting to transform transaction data fields before sending to PSPs or storing them in the platform.](https://documentation.ixopay.com/manual/docs/connector/advanced-configuration/field-expression-language)## [ Network Tokenization Setting Configure network tokenization behavior on connectors to control how card network tokens are used when processing transactions.](https://documentation.ixopay.com/manual/docs/connector/advanced-configuration/network-tokenization-setting)
