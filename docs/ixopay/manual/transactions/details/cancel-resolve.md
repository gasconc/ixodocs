# Cancel/Resolve
## Cancel Transaction[​](https://documentation.ixopay.com/manual/docs/transactions/details/cancel-resolve#cancel-transaction "Direct link to Cancel Transaction")
Depending on the given Permissions and prerequisites, Users can cancel transactions manually (e.g. in case a Transaction gets stuck). This can be done by using the **Cancel** Action Buttons in the Transaction Details.
To cancel a Transaction, the Transaction has to fulfill the following condition:
  * The **Transaction status** has to be Open (except Pending_Postback)

Once confirmed the Transaction will be cancelled and shown as such in the Transaction Status.
