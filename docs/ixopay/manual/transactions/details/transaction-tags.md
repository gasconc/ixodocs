# Transaction Tags
note
Please contact your Customer Success Manager or our Support Team in the [IXOPAY Customer Experience Portal](https://ixopay.my.site.com/support) to setup tags in your configuration.
Once enabled, you can give [Roles](https://documentation.ixopay.com/manual/docs/system-setup/roles) the permission to add and remove Tags to Transactions (see Permission tag). This can be done in the Transaction Details under the Transaction Data Section.
  1. Click **Add Tag** (see Adding Tag)
  2. Select **fraud** in the dropdown selection for Tenant Tag (see Adding Fraud Tag)
  3. Click **Add Tag**

The **fraud tag** will show up in the Transaction Details (see Fraud Tag in Transaction Details) and can be removed by clicking the **x** directly on the Tag and confirming the removal (see Remove Tag Confirmation).
