## Create and Export customized Reports[​](https://documentation.ixopay.com/manual/docs/transactions/data-export#create-and-export-customized-reports "Direct link to Create and Export customized Reports")
The IXOPAY platform enables you to create and export customized Transaction Reports, by following these steps:
  1. Navigate to the **Transactions** section, sub-section **Data Export**.
  2. Click on **+ New Transactions Export**
  3. Select the **File Type** and **Filter** your Transactions according to your requirements (at least one **Merchant, Provider or Connector and a Date Range must be selected**)
  4. Click on **Customize Columns** to **tick/untick** columns and **rearrange** the order of columns **via drag and drop** for the created Export
  5. Once finalized, click **Generate** for the requested Transaction Export to be created. The report will be in status **Waiting** depending on the amount of data to be collected (Up to 10 minutes, reports up to 10k Transactions are usually ready within 1 minute)

Refreshing the page is possible during generation and will show you processed percentage and count of rows. During generation you can **cancel** the Transaction Export anytime.
The customized Report will be shown in the Data Export Overview in status Done for **Download** in the selected data format or **Deletion**.
