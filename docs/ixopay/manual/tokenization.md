# Tokenization
In this section of the manual, you’ll find tools to tokenize and vault payment details— reduce PCI scope, manage tokens, and control their lifecycle across your [IXOPAY platform](https://www.ixopay.com) flows.
## [ Customer Profiles Store and manage customer data and payment instruments using Customer Profiles to enable tokenized repeat purchases via API or the platform interface.](https://documentation.ixopay.com/manual/docs/tokenization/customer-profiles)## [ Account Updater The IXOPAY Account Updater automatically refreshes stored card data for Mastercard, Visa, and Discover accounts to prevent expired-card declines.](https://documentation.ixopay.com/manual/docs/tokenization/account-updater)## [ Network Token Services IXOPAY Network Token Services use scheme-issued tokens from Visa and Mastercard to keep card data current, reduce PCI scope, and lower processing fees.](https://documentation.ixopay.com/manual/docs/tokenization/nt)
