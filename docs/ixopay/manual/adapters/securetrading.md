# Trust Payment / Secure Trading
Configure the following parameters for the Connector (see Connector Config - SecureTrading):
  1. Fill in the mandatory **Username** - Merchant account username
  2. Fill in the mandatory **Password**
  3. Fill in the mandatory **API Secret**
  4. Fill in the **Extra Data: CheckResponseSecurity** — true, false
  5. Fill in the **Extra Data: paymentPagesPassword**
  6. Fill in the **Extra Data: paymentPagesHashAlgo**
  7. Fill in the mandatory **Extra Data: wsUsername**
  8. Fill in the mandatory **Extra Data: wsPassword**
  9. Fill in the **Extra Data: notificationPassword**
  10. Fill in the **Extra Data: notificationHashAlgo**

## Credit card[​](https://documentation.ixopay.com/manual/adapters/securetrading#credit-card "Direct link to Credit card")
Configure the following parameters for the Connector (see Connector Detail Overview - Trust Payments Creditcard - Vault Configuration):
  1. Select the mandatory **Environment** : Test, Live
  2. Select the mandatory **Gateway Region** : Europe, US
  3. Fill in the mandatory **Username**
  4. Fill in the mandatory **Password**
  5. Fill in the mandatory **Live Site Reference**
  6. Fill in the optional **Test Site Reference**
  7. Select the optional **Descriptor** type - None, Use Static value (next field), Dynamic Descriptor
  8. Fill in the optional **Static Descriptor**
