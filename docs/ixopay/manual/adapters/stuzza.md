# Stuzza
Configure the following parameters for the Connector (see Connector Config - Stuzza):
  1. Fill in the mandatory **User ID**
  2. Fill in the mandatory **Merchant PIN**
  3. Fill in the mandatory **Extra Data: beneficiaryBIC** — ISO Business Identifier Code (ISO 9362)
  4. Fill in the mandatory **Extra Data: beneficiaryName**
  5. Fill in the mandatory **Extra Data: beneficiaryIBAN** — International Bank Account Number (ISO 13616)
  6. Fill in the mandatory **Extra Data: dynamicDescriptor:** true, false
