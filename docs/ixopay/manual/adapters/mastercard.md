# Mastercard
## Mastercard Creditcard[​](https://documentation.ixopay.com/manual/adapters/mastercard#mastercard-creditcard "Direct link to Mastercard Creditcard")
Configure the following parameters for the Connector (see Connector Detail Overview - Mastercard Creditcard - Vault Configuration):
  1. Select the **Environment:** Live, Test
  2. Fill in the mandatory **API Username**
  3. Fill in the mandatory **API Password**
  4. Fill in the mandatory **Merchant ID**
  5. Fill in the optional **Merchant Website URL (required for 3D-Secure)**
