# Selfpay
## Generic[​](https://documentation.ixopay.com/manual/adapters/selfpay#generic "Direct link to Generic")
Configure the following parameters for the Connector (see Connector Config - Selfpay):
  1. Fill in the mandatory **Username**
  2. Fill in the mandatory **Password**
  3. Fill in the mandatory **API Secret**
