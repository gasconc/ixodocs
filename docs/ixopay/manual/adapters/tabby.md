# Tabby
## Tabby[​](https://documentation.ixopay.com/manual/adapters/tabby#tabby-1 "Direct link to Tabby")
Configure the following parameters for the Connector (see Connector Config - Tabby):
  1. Fill in the mandatory **Secret Key**
  2. Fill in the mandatory **Public Key**
  3. Fill in the mandatory **Extra Data: merchant_code**
  4. Fill in the mandatory **Extra Data: webhook_signature**
