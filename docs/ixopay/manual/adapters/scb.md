# SCB
Configure the following parameters for the Connector (see Connector Config - SCB):
  1. Fill in the mandatory **API Key (Client ID)**
  2. Fill in the mandatory **Shared Secret (Client Secret)**
  3. Fill in the mandatory **Extra Data: merchantAccountId** — Unique ID to identify the merchant, provided by PSP
  4. Fill in the mandatory **Extra Data: subAccountId**
  5. Fill in the mandatory **Extra Data: certificate**
