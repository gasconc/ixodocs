# Kassa Compleet
Configure the following parameters for the Connector (see Connector Config - Kassa Compleet):
  1. Fill in the mandatory **Username** Merchant’s account name.
  2. Fill in the mandatory **Password** Merchant’s account password.
  3. Fill in the mandatory **API Secret**
  4. Fill in the mandatory **Extra Data: issuerRetrieveAllowedOrigins**
