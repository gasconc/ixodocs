# Silverflow
## Silverflow Creditcard[​](https://documentation.ixopay.com/manual/adapters/silverflow#silverflow-creditcard "Direct link to Silverflow Creditcard")
Configure the following parameters for the Connector (see Connector Detail Overview - Silverflow Creditcard - Vault Configuration):
  1. Select the mandatory **Environment:** Live, Test
  2. Fill in the mandatory **Merchant Acceptor Key**
  3. Fill in the mandatory **Username**
  4. Fill in the mandatory **Password**
