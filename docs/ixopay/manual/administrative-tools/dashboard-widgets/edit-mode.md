# Add/Customize Dashboards
You can add as many dashboards as you want and customize them in order to optimize that analysis of different aspects of you payment stack.
Select **Add new Dashboard** from the Dashboard drop-down to add a new dashboard and customize it.
![Add new Dashboard](https://documentation.ixopay.com/manual/assets/ideal-img/add-new-dashboard.e9b0efa.1138.png)Add new Dashboard![New Dashboard](https://documentation.ixopay.com/manual/assets/ideal-img/new-dashboard.c2ad7ea.1280.png)New Dashboard
## Customize Dashboard[​](https://documentation.ixopay.com/manual/docs/administrative-tools/dashboard-widgets/edit-mode#customize-dashboard "Direct link to Customize Dashboard")
Click on **Customize** to customize a new or existing dashboard. The following options are available:
  * [Add/Remove Widget](https://documentation.ixopay.com/manual/docs/administrative-tools/dashboard-widgets/edit-mode#addremove-widget)
  * [Configure Widget](https://documentation.ixopay.com/manual/docs/administrative-tools/dashboard-widgets/edit-mode#configure-widget)
  * [Reorder/Resize Widget](https://documentation.ixopay.com/manual/docs/administrative-tools/dashboard-widgets/edit-mode#reordering--resizing-widgets)
  * [Rename Dashboard](https://documentation.ixopay.com/manual/docs/administrative-tools/dashboard-widgets/edit-mode#rename-dashboard)
  * [Clone Dashboard](https://documentation.ixopay.com/manual/docs/administrative-tools/dashboard-widgets/edit-mode#clone-dashboard)
  * [Share Dashboard and edit shared Dashboards](https://documentation.ixopay.com/manual/docs/administrative-tools/dashboard-widgets/edit-mode#share-dashboard)
  * [Delete Dashboard](https://documentation.ixopay.com/manual/docs/administrative-tools/dashboard-widgets/edit-mode#delete-dashboard)

Click on **Save** after customizing the dashboard to save your changes, or click on **Cancel** to revert back to the previously saved state.
![Edit Mode](https://documentation.ixopay.com/manual/assets/ideal-img/edit-mode.bb81fce.1280.png)Edit Mode
## Add/Remove Widget[​](https://documentation.ixopay.com/manual/docs/administrative-tools/dashboard-widgets/edit-mode#addremove-widget "Direct link to Add/Remove Widget")
Click **Add Widget** to add a new widget to the dashboard. A list of available widgets is displayed. Click on a widget in the list to add it to the dashboard.
Remove a widget by clicking the **X** icon on the widget itself.
![Widget Selection](https://documentation.ixopay.com/manual/assets/ideal-img/widget-selection.e6fe7a2.932.png)Widget Selection![Remove Widget](https://documentation.ixopay.com/manual/assets/ideal-img/remove-widget.b81a9b7.1280.png)Remove Widget
## Configure Widget[​](https://documentation.ixopay.com/manual/docs/administrative-tools/dashboard-widgets/edit-mode#configure-widget "Direct link to Configure Widget")
Click on the **gear** icon in a widget to configure it. You can:
  * Change the name of the widget displayed in the dashboard
  * Override the global filter
  * Define additional settings depend on the widget type

When applying changes to a a widget's configuration or filter, the data displayed in the widget is reloaded, providing instant visual feedback.
![Configure Widget](https://documentation.ixopay.com/manual/assets/ideal-img/configure-widget.d861402.926.png)Configure Widget
## Reordering & Resizing Widgets[​](https://documentation.ixopay.com/manual/docs/administrative-tools/dashboard-widgets/edit-mode#reordering--resizing-widgets "Direct link to Reordering & Resizing Widgets")
**Reorder** the widgets in your dashboard via drag & drop. Surrounding Widgets are reordered automatically. Resize a widget by dragging the bottom right corner to the desired size.
![Reorder](https://documentation.ixopay.com/manual/assets/ideal-img/dashboard-reorder.889c225.1280.png)Reorder![Resize](https://documentation.ixopay.com/manual/assets/ideal-img/dashboard-resize.45cd7d4.1280.png)Resize
## Rename Dashboard[​](https://documentation.ixopay.com/manual/docs/administrative-tools/dashboard-widgets/edit-mode#rename-dashboard "Direct link to Rename Dashboard")
Click **Edit** and enter a name in the **Dashboard Name** field to rename the entire dashboard (see Rename).
![Rename](https://documentation.ixopay.com/manual/assets/ideal-img/rename.c7b3a60.1280.png)Rename
## Clone Dashboard[​](https://documentation.ixopay.com/manual/docs/administrative-tools/dashboard-widgets/edit-mode#clone-dashboard "Direct link to Clone Dashboard")
Click **Clone Dashboard** to clone a dashboard (e.g. to adapt the dashboard to a new use case without starting from scratch). A copy of the current dashboard is created and opened **without saving** the original dashboard first.
![Cloned Dashboard](https://documentation.ixopay.com/manual/assets/ideal-img/cloned-dashboard.b090715.1280.png)Cloned Dashboard
## Share Dashboard[​](https://documentation.ixopay.com/manual/docs/administrative-tools/dashboard-widgets/edit-mode#share-dashboard "Direct link to Share Dashboard")
Click on **Share Dashboard** to share the current dashboard. Sharing dashboards allows you to provide other users in you enterprise with access to general reports (like top merchants, transaction stats or chargeback rate). Once shared, the dashboard is automatically available to all users.
## Editing Shared Dashboards[​](https://documentation.ixopay.com/manual/docs/administrative-tools/dashboard-widgets/edit-mode#editing-shared-dashboards "Direct link to Editing Shared Dashboards")
Users can only edit or delete a shared dashboard if they have the necessary permissions. Users without the required permission may customize the [Dashboard](https://documentation.ixopay.com/manual/docs/administrative-tools/dashboard-widgets/dashboard-widgets) but cannot save their changes. To save changes, users need to clone the dashboard and edit the copy instead.
## Delete Dashboard[​](https://documentation.ixopay.com/manual/docs/administrative-tools/dashboard-widgets/edit-mode#delete-dashboard "Direct link to Delete Dashboard")
Click on **Delete Dashboard** to delete a dashboard. Confirm the prompt. The dashboard is deleted and the next dashboard in the list is displayed. If you deleted the last dashboard, you can create a new Dashboard.
![Refresh](https://documentation.ixopay.com/manual/assets/ideal-img/dashboard-delete.c98e15a.630.png)Refresh
