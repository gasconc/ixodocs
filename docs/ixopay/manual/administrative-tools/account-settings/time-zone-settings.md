# Timezone Settings
The timezone configured for your account is applied to all events, transaction details and reporting within [IXOPAY](https://www.ixopay.com). To set up or change your account's timezone:
  1. Either select the **Timezone** in the top bar or select the drop-down menu next to your user name in the top right corner (see Account Settings).
  2. **Select** the time zone from the drop-down.
  3. Click **Save Changes**.

## User Settings[​](https://documentation.ixopay.com/manual/docs/administrative-tools/account-settings/time-zone-settings#user-settings "Direct link to User Settings")
