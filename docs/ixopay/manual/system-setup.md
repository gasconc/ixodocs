# System Setup
In this section of the manual, you’ll find foundational setup for your [IXOPAY platform](https://www.ixopay.com) environment— configure accounts, roles, environments, and core settings to get operational quickly and safely.

communication details for critical incident response.](https://documentation.ixopay.com/manual/docs/system-setup/service-contacts)## [ Currency Conversion Rates Manage currency conversion rates with daily ECB updates and custom tolerances for multi-currency reporting and risk rules.](https://documentation.ixopay.com/manual/docs/system-setup/currency-conversion-rates)## [ API Keys Create and HPP CSP Configuration Use HPP CSP Configuration to control which scripts your Hosted Payment Page (HPP) is allowed to load and hostnames, supporting TLS with SHA-2 and ECDSA or RSA.](https://documentation.ixopay.com/manual/docs/system-setup/ssl-configuration)
