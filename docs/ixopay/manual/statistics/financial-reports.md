# Financial Reports
## Video: Creating Financial Reports[​](https://documentation.ixopay.com/manual/docs/statistics/financial-reports#video-creating-financial-reports "Direct link to Video: Creating Financial Reports")
![](https://documentation.ixopay.com/manual/assets/ideal-img/preview-vimeo-statistics-reports.54d6918.1300.png)
Learn how to easily create financial reports.
The **Statistics** section provides you with options to generate extensive **Financial Reports** and **Pivot Tables** directly in the [IXOPAY platform](https://www.ixopay.com).
To generate a report:
  1. Select **Statistics** in the menu and select **Financial Reports**.
  2. Select the type of **report** to be generated.
    1. **Merchant Overview** either **from Merchant perspective** or **with all data:** provides information about total Transaction Volume and Fees (only if [Fee Entity](https://documentation.ixopay.com/manual/docs/connector/fee-management/fee-entities) is marked as Merchant Visible) of the Merchant (see Merchant Overview) including the **Overall (Sum of all Connectors)** Volume at the bottom of the report.
    2. **Provider Summary** either **by detected country** or **by customer billing country** : provides an overview about the total Transaction Volume by Currency/Region per Provider (see Provider Summary).
  3. Select the **Date range** for the report.
  4. Select a **Tenant, Merchant** and **Connector** for which the report should be generated.
  5. Click **Generate**.
