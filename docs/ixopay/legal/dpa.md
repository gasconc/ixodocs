# Dpa

## **Current Data Processing Addendum: Effective Feb 13, 2026**
This Data Processing Addendum (DPA) outlines IXOPAY’s obligations and Customer responsibilities under global data protection laws, including GDPR, UK GDPR, LGPD, CPRA, and U.S. state privacy laws. It covers roles of the parties, processing instructions, security measures, subprocessors, data subject rights, breach notifications, and cross-border transfer mechanisms such as Standard Contractual Clauses.
