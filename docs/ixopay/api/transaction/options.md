# Options
Access additional information provided by certain adapters to enhance your payment integration.
For example, some adapters offer a bank list for EPS transactions, allowing you to display the bank selection directly on your checkout site.
See the [adapter specific](https://documentation.ixopay.com/adapters/simulator) page for further information about available Options request methods.
