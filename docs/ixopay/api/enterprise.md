# Enterprise
This section provides information on APIs that are only available to our enterprise customers. We have included detailed information on how to access and use these APIs, along with guidelines for integration and best practices. Whether you're a new or existing enterprise customer, this page provides a wealth of information on how to take advantage of our advanced APIs to achieve your business goals. Explore this page to learn more about our enterprise APIs and how they can benefit your organization.
## [ PCI Transaction API This API enables PCI-compliant merchants to send transactions containing card data directly.](https://documentation.ixopay.com/api/pci/pci-transaction-api)## [ Customer profiles API The customer profiles API allows you to store customer data and multiple payment instruments (for multiple payment methods/connectors) for your customers.](https://documentation.ixopay.com/api/customer-profiles/customer-profiles-api)## [ Reconciliation API Using our Reconciliation API, you may retrieve reconciliations of transactions within a given date range.](https://documentation.ixopay.com/api/reconciliation/reconciliation-api)## [ Settlement API Using our Settlement API, you may retrieve settlements of your merchant within a given date range. The response will contain all settlement files of the specified date range.](https://documentation.ixopay.com/api/settlement/settlement-api)## [ Provisioning API The provisioning API allows you to manage your IXOPAY platform entities. You can create, read, update tenants, merchants, merchant users and connectors.](https://documentation.ixopay.com/api/provisioning/provisioning-api)
