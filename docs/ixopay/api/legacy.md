# Legacy
This section lists APIs that are no longer recommended for use and should be avoided. These APIs have been deprecated due to changes in our system or updated API offerings, and we encourage all customers to switch to the latest version of our APIs as soon as possible. However, we understand that some customers may have already implemented their integration using these deprecated APIs and need to continue using them for the time being. For this reason, we have included detailed information on these APIs to help customers who have already integrated with them to continue using them until they can switch to the latest version.
