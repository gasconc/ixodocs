# Finance Wide
This page provides an overview of the payments methods provided by the Finance Wide adapter in the IXOPAY platform. It also includes a full list of all configuration options available to you when integrating Finance Wide within your payments landscape, as well as an overview of the parameters required when submitting a transaction via IXOPAY's API.
## Payment Methods[​](https://documentation.ixopay.com/adapters/finance-wide#payment-methods "Direct link to Payment Methods")  
| Payment Method  | Transaction Flows  | Transaction Types  |  
| --- | --- | --- |  
| Creditcard  | iFrame Form & payment.js Integration  | All (no recurring, CVV must be present)  |  
## Additional Parameters[​](https://documentation.ixopay.com/adapters/finance-wide#additional-parameters "Direct link to Additional Parameters")
[3D-Secure Verification](https://documentation.ixopay.com/docs/reference/features/3d-secure) parameters are ignored entirely, however the underlying gateway manages and changes the behavior of 3D-Secure requirements on demand.
