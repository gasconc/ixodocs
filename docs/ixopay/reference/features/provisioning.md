# Provisioning
The [Provisioning API](https://documentation.ixopay.com/api/provisioning/provisioning-api) is a component of the [IXOPAY platform](https://www.ixopay.com) that provides programmatic access to create and manage merchants, merchant users, and connectors. This API is designed to facilitate the automation of these tasks, reducing the need for manual intervention and increasing efficiency.
The Provisioning API allows you to:
  * Automate the creation and configuration of merchants
  * Manage merchant users and their access permissions
  * Set up and configure connectors to various payment gateways
