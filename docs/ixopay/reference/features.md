# Features
Welcome to the features section of our [IXOPAY platform](https://www.ixopay.com) reference documentation. This section provides comprehensive information on the diverse features available in our platform.
The IXOPAY platform offers a wide array of features, including 3-D Secure authentication, scheduling transactions, risk check mechanisms, dynamic currency conversion (DCC), customer profiles, business intelligence data sourcing, and flexible payment method selection.
Please navigate to the specific articles to gain detailed insights into each feature:
