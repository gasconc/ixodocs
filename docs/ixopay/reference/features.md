# Features
Welcome to the features section of our [IXOPAY platform](https://www.ixopay.com) reference documentation. This section provides comprehensive information on the diverse features available in our platform.
The IXOPAY platform offers a wide array of features, including 3-D Secure authentication, scheduling transactions, risk check mechanisms, dynamic currency conversion (DCC), customer profiles, business intelligence data sourcing, and flexible payment method selection.
Please navigate to the specific articles to gain detailed insights into each feature:
## [ 3-D Secure 5 items](https://documentation.ixopay.com/docs/reference/features/3d-secure)## [ Scheduler The scheduler enables you to perform recurring debits without any further intervention.](https://documentation.ixopay.com/docs/reference/features/scheduler)## [ Risk checks This section provides detailed information on the various risk checking tools](https://documentation.ixopay.com/docs/reference/features/risk-checks)## [ Dynamic currency conversion (DCC) Dynamic Currency Conversion (DCC) is a feature that allows customers to see the cost of transactions in their local currency, providing a more familiar and transparent purchasing experience.](https://documentation.ixopay.com/docs/reference/features/dcc)## [ Customer profiles The customer profiles API enables you to store customer data and multiple payment instruments (for multiple payment methods/connectors) for your customers.](https://documentation.ixopay.com/docs/reference/features/customer-profiles)## [ Pay-by-Link The Pay-by-Link feature offers the possibility to generate a secure URL leading to a customized payment page for customers.](https://documentation.ixopay.com/docs/reference/features/pay-by-link)## [ BI data source 1 items](https://documentation.ixopay.com/docs/reference/features/bi-data-source)## [ Payment method selection 1 items](https://documentation.ixopay.com/docs/reference/features/payment-method-selection)## [ Provisioning 1 items](https://documentation.ixopay.com/docs/reference/features/provisioning)## [ Tokenization & PCI 1 items](https://documentation.ixopay.com/docs/reference/features/tokenization/pci-data-migration)
These articles aim to provide you with a profound understanding of how to utilize these features to the fullest, thereby optimizing your payment processes with the IXOPAY platform.
