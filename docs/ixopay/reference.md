# Overview
Welcome to the [IXOPAY platform](https://www.ixopay.com) reference documentation. This section provides in-depth information on the payment platform and is intended for developers and technical users who want to get a deeper understanding of the platform's integration options, concepts, and features.
Guide
If you are just starting out with the IXOPAY platform, we recommend that you read the [Guides](https://documentation.ixopay.com/docs/guides) section first. This section provides step-by-step instructions for integrating the payment platform into your website or application.
## Structure of the documentation[​](https://documentation.ixopay.com/docs/reference#structure-of-the-documentation "Direct link to Structure of the documentation")
This documentation is organize into the following sections:
  * [Integration](https://documentation.ixopay.com/docs/reference/integration): contains articles about integration options.
  * [Concepts](https://documentation.ixopay.com/docs/reference/concepts): provides detailed information on IXOPAY platform specific concepts such as transaction types or transaction indicators.
  * [Features](https://documentation.ixopay.com/docs/reference/features): contains reference information on additional [IXOPAY](https://www.ixopay.com) features that tie in to the payment process.
  * [Appendix](https://documentation.ixopay.com/docs/reference/appendix): contains various listings, including error codes and other reference information.

In conclusion, the IXOPAY platform reference documentation provides detailed information on the payment platform, including integration optsion, concepts, and additional features. We hope you find this documentation helpful, and if you have any questions, please don't hesitate to contact our Support Team.
