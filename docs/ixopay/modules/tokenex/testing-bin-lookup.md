# Testing BIN Lookup
The TokenEx TEST environment has a BIN file that is a snippet of BINs from the full BIN file. The allows customers to test the functionality of our BIN lookup capabilities using test account numbers that fall into one of the ranges that are included in the test file. **Refer to the table below for valid test account number ranges.**
