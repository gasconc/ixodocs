# Device Services (P2PE)
The **Device Services** API is used for point-to-point encryption (P2PE) tokenization requests. Use of the **Device Services** API requires configuration of an Encryption Profile with the ability to decrypt data encrypted on a card payment terminal or pin-pad device.
