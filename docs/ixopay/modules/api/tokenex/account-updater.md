# Account Updater
The Account Updater API is batch API used for getting updated card metadata such as the expiration date or account number (PAN) for a card.
Account Updater API enables credential-on-file merchants to securely access updated account information, facilitating a more seamless payment process and helps merchants reduce the risk of declined transactions.
